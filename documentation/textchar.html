Class TextChar

Class for a single character inside of a text object. An instance of this class holds positioning, rendering, and animation information for an individual character.

Property List

animations[] - The array of current applied Animation instances. To start an animation on a textchar, just push it into this array.
assocSymbol - The symbol that this textchar is based on.
blockSize[] - The array containing the x and y dimensions of each rendered block. ([0] = width [1] = height.)
color - The hex-color code for the letter. (Note that this value is a string that must start with the hashtag "#".) Default is "#000000".
opacity - The opacity of each rendered block. Default is 1.
positionMatrix[] - The matrix containing the x and y coordinates for each block of the letter.
textContainer - The Text object that contains the current textchar.

render(dt, context)

This function should be called by the Text that contains the textchar. It handles rendering out each individual block to the canvas.

Input Parameters

dt - Change in time, or amount of time in milliseconds since the last time this function was called (necessary for any animations that are running.)

context - The canvas 2d context you wish to render the text to.
