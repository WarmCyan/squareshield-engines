Property List

clickable - Boolean value representing whether the object will respond to a mouse click or not.
color - String hexidecimal color code for the square (should start with '#')
damagable - Boolean value representing whether the object can be damaged or not.
health - Remaining health, if damage is enabled and this property is below zero, it will be removed from the world.
name - Small identifying string so that it can be identified in the object console
physics - Physics object that contains all of the physical properties of this object
